Como realizar la consulta

×

Para realizar la busqueda de un registro o diversos registros puede hacerlo mediante las siguiente opciones:

  • Opción "Dia": Ingresar ID del sensor y/o fecha del dia especifico a consultar.
  • Opción "Mes/Año": Ingresar ID del sensor, mes, año. Puede ingresar solamente uno de los datos o combinacion de dos o tres
  • Opción "Rango": Ingresar ID del sensor y/o rango de fechas.

En cada una de las opciones se puede ingresar solamente el ID o uno de los datos que se muestran, al igual que se pueden ingresar tanto ID como el resto de datos de la opcion seleccionada.
